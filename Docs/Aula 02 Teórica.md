# O que significa MER?
- M -> Modelo
- E -> Entidade
- R -> Relacionamento
---

- Entidade -> Ela é um elemento e por obrigação ela deve ter suas caracteristicas(atributos). Ela é representado por um retangulo.
## Atributos da Entidade
- Atributo Identificador -> Eu preciso encontrar algum dado que faça eu identificar um dado único, que não se repete em nenhum cadastro.
- Exemplos: Cliente -> CPF, Nome, Telefone, RG. // Produto -> Nome, Cor, Tamanho, Codigo
