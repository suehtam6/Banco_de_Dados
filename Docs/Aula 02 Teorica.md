# O que significa MER?
- M -> Modelo
- E -> Entidade
- R -> Relacionamento
- Entidade -> Ela é um elemento e por obrigação ela deve ter suas caracteristicas(atributos). Ela é representado por um retangulo.
---

## Atributos da Entidade
- Atributo Identificador -> Eu preciso encontrar algum dado que faça eu identificar um dado único, que não se repete em nenhum cadastro.
- Exemplos: Cliente -> Nome, Telefone, Id. // Produto -> Nome, Cor, Tamanho, Id.

- Atributo Simples -> Ela é ligada a um processamento simples de informação.
- Exemplos: Pode ser utilizado o E-mail

- Atributo Composto -> Ele é composto por outros atributos, para ser formado.
- Exemplos: Endereço, pois é feito de varias etapas tipo a Rua, o número da casa, a cidade, o Estado e o País.

- Atributo MultiValorado -> os software MultiValorados normalmente usam * ou (1,n). O Atributo MultiValorado pode receber mais do que um valor para ele mesmo.
- Exemplos: Cliente -> Id, Nome, Telefone(Pode receber mais do que um número), E-mail(Pode receber mais que um E-mail). Normalmente o software irá deixar salvo
Desta maneira Telefone(1,n).
